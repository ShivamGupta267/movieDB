# movieDB
Fetching database of movies for a demo project.

# Given API `https://dummyapi.online/api/movies` is not working.  
Using **OMDb API** instead.

## Setup Instructions:
1. Clone the repo  
2. Run `npm install`  
3. Start the project using `npm run dev`

## Tech Stack:
- React + Vite
- Fetch API
- OMDb API
